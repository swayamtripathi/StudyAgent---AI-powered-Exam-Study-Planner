# StudyAgent – AI-powered Exam Study Planner

StudyAgent is an AI-powered web application that helps students generate personalized study schedules for upcoming exams.

The application takes user inputs such as subjects, exam date, and available study hours, and uses an AI model to generate a structured day-by-day study plan.

**Sanskar Tripathi**

---

## Problem

Students often struggle to create an effective study schedule before exams.  
Manual planning can be inefficient and may not allocate time properly across subjects.

StudyAgent solves this problem by using AI to automatically generate a personalized study plan.

---

## Features

- AI-generated study schedules
- Interactive study hours visualization
- Study progress tracker
- Downloadable study plan
- Clean and simple user interface

---

## How It Works

The workflow of StudyAgent is shown below:

User Input  
↓  
Streamlit Web Interface  
↓  
Planner Logic (Python)  
↓  
Groq AI Model  
↓  
AI Generates Study Plan  
↓  
Study Plan + Visualization Displayed
